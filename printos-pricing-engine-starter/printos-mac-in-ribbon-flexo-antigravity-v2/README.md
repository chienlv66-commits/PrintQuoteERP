# PrintOS - Mác in Riboon & In Flexo v2

Bản v2 đã chốt theo phản hồi vận hành:

- Riboon: nền bệt giữ nguyên như Excel, UI vẫn chọn 0/1 nhưng không tác động công thức.
- Riboon: phí khác mặc định 30.000, admin được sửa.
- Flexo: phí ra fim mặc định 25.000, admin được sửa.
- Flexo: số mã trên 1 bài in luôn >= 1, mặc định 1 nếu không nhập.

Đưa file `.md` cho Google Antigravity trước, sau đó dùng `.ts` và `.json` để cập nhật module `src/pricing/modules/mac-in/`.
